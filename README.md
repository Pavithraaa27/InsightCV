# 📄 CVInsight — Resume Screening & Candidate Analysis Platform

CVInsight is a web application that helps evaluate resumes against a job description. Users can upload resumes, provide a job description, and receive structured feedback to assist in candidate evaluation and comparison.

---

## ✨ Features

- Upload resumes
- Enter a job description
- Resume feedback based on the provided job description
- Candidate comparison
- Top candidate ranking
- Resume analytics dashboard
- Responsive user interface

---

## 🛠 Tech Stack

| Layer | Technology |
|--------|------------|
| Frontend | React, TypeScript, Vite |
| Styling | Tailwind CSS |
| Backend / Database | Supabase |
| Deployment | Vercel (optional) |

## 🚀 Getting Started

### Install dependencies

```bash
npm install
```

### Run the application

```bash
npm run dev
```

Open

```
http://localhost:5173
```

---

## 📋 Workflow

1. Upload one or more resumes.
2. Enter the job description.
3. The application processes the uploaded resumes.
4. Candidate feedback and comparison are generated.
5. View candidate rankings and analytics.
